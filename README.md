# AI Image Detection Bypass

对抗 AI 生成图片检测工具的简易脚本。通过 FFT 频谱匹配、相机管线模拟、噪声注入等方法，修改图片的统计特征以绕过 AI 图像检测器。

## 原理

AI 图像检测器主要依据以下特征判断图片是否为 AI 生成：

- **频率域**：AI 生成图的幅度谱偏离自然图像的 1/f^α 幂律分布
- **像素域**：噪声水平异常低
- **物理痕迹**：缺少真实相机传感器的 Bayer、镜头色差、JPEG 压缩等处理痕迹
- **元数据**：缺少 EXIF 相机信息

本工具针对性地修改这些特征。

## 安装

```bash
pip install -r requirements.txt
```

## 用法

```bash
python bypass_detector.py input.png output.jpg
```

参数在脚本顶部 `PARAMS` 字典中调节。

## 方法

| 模块 | 作用 |
|------|------|
| FFT 频谱匹配 | 将幅度谱掰向 1/f^α 自然分布（LAB 色彩空间，只改 L 通道） |
| 相机管线 | 模拟 Bayer→色差→暗角→传感器噪声→坏点→JPEG 全链路 |
| 高斯噪声 | 注入高斯噪声打破过干净的像素分布 |
| 像素扰动 | 均匀分布微扰 |
| 自动白平衡 | 灰世界假设（默认关闭，会破坏暖色调） |
| 假 EXIF | 写入随机相机元数据 |
